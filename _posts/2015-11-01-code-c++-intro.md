#### 语言类型
+ c++容忍隐士的类型转换，所以是一种种弱类型语言
+ c++编译时进行类型检查，程序运行时不能改变其结构，所以是静态语言
#### 版本问题
+ c++11 版本，以前的C++代码风格极其混乱。
+ 代码使用C++11的风格，以前的不要用了。


.h文件和.cpp文件  后缀名是为了给程序员看的，没有什么用。  
C++中，变量，函数使用之前需要声明，头文件k就是起声明的作用。
