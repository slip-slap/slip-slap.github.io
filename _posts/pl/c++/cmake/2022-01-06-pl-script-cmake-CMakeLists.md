####  Project Command

```cmake
project(GM LANGUAGES CXX)
```
1. This command set the name of the project, and also set a set of variables.
- PROJECT\_SOURCE\_DIR, \<PROJECT-NAME\>\_SOURCE\_DIR
- PROJECT\_BINARY\_DIR, \<PROJECT-NAME\>\_BINARY\_DIR
